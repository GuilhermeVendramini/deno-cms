# Deno CMS AND API

### Requirements:

* Denon
* Mongodb

### Run:

```shell
denon install
```

```shell
denon start
```

### Default user login:

**email**: admin@admin.com
**password**: 12345678

## Features

**CMS**
* User roles permissions
* User register/login
* Basic Page content type (CRUD)

**API**
* User register/login
	* POST USER REGISTER
	* POST USER LOGIN
	* GET ONE
	* GET ALL
* Basic Page content type 
	* GET ONE
	* GET ALL
